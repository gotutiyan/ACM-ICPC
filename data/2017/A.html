問題: 太郎君の買い物

問題の概要

長さnの数列から、任意の２つの要素を取り出せる。この要素の和がm以下の範囲での最大値を求める。

解説

まず最初に、nとmが0 0になるまで入力を受け取るという部分は、以下のように書けます。


int n,m;
while(cin>>n>>m && n && m){
  //処理
}

さて、計算量の話として、今回nは1000を超えないので２重ループを回しても計算量は106となり間に合いそうであることが分かります。

最初、出力用の変数ansを用意し、0で初期化します。ここで２重ループのループ変数をi,jとすれば、i≠j の時、数列のi番目とj番目の要素の和を見て、それがm以下であればansとの最大値を取ります。これを繰り返すことで、答えが求まります。i≠jとしているのは、もしi=jを許すと同じ商品を２つ買うことになり、問題の設定に反するからです。
最後に、ansが0のままなら、求める組み合わせが見つからなかったということなので、"NONE"を出力します。

この解法では、考えられる２つの要素の取り方を全て試すことになるので、全探索という手法になります。全探索は計算量こそ多いですが、実装やアルゴリズムが簡単なので小さい要素数に対して絶大な効果を発揮します。


#include <iostream>
#include <vector>
using namespace std;

int main (){
  int n,m;
  while(cin>>n>>m && n && m){
    int ans=0;
    vector<int> v(n);
    for(int i=0;i<n;i++)cin>>v[i];

    for(int i=0;i<n;i++){
      for(int j=0;j<n;j++){
        if(i!=j && v[i]+v[j]<=m){ //同じ要素でなく、かつ和がm以下なら
          ans=max(ans,v[i]+v[j]); //ansとの最大値を取る
        }
      }
    }
    if(ans==0)cout<<"NONE"<<endl;
    else cout<<ans<<endl;
  }
}
