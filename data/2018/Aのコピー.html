問題: 所得格差

問題の概要

数列が与えられるので、その全要素の平均以下の値を持つ要素数を求める。

解説

入力を数列に格納し、一旦全ての値を足して平均を取ったあと、条件を満たす要素数を求めます。
求まる平均は必ずしも整数であるとは限らないので、合計を貯めるための変数はdoubleなどの浮動小数点型にすることに注意です。

また、「0」が来たら終了という処理は、while文を使って以下のように書けます。これはC++においてif(0)のように条件式に0が入れば、その条件式は必ずfalseになり実行されないことを利用しています。

int n;
while(cin>>n && n){
    //処理

}

これを踏まえて、解答例は以下のようになります。

#include <iostream>
#include <vector>
using namespace std;

int main (){
    int n;
    while(cin>>n && n){
        vector<int> v(n);
        double sum=0;
        for(int i=0;i<n;i++){//入力を受けてまずは合計を求める
            cin>>v[i];
            sum+=v[i];
        }
        sum/=n;//平均を出して

        int ans=0;
        for(int i=0;i<n;i++){
            if(sum>=v[i])ans++;//i番目について、平均以下ならansを増やす
        }
        cout<<ans<<endl;
    }
}

計算量はO(n)で、十分に間に合います。
